Real-Time Face Detection

Summary: An industrial-grade access control solution using YOLOv8 for real-time face detection, optimized for edge devices and harsh conditions.

Expanded Description:
This system was trained and optimized with YOLOv8 and PyTorch, leveraging extensive augmentation and quantization for deployment in real-world environments. It ensures reliable detection under low light, varied angles, and high throughput scenarios. Exported with ONNX/TensorRT, it achieves low-latency inference on embedded devices.

Results: Achieved 95% accuracy in complex conditions while maintaining real-time performance for critical access control systems.

Stack & Tools: YOLOv8, Ultralytics, PyTorch, OpenCV, Edge Optimization, ONNX, TensorRT, Jetson devices.

Status: Deployed prototype. Current phase: expansion for multi-camera environments and integration with biometric authentication layers.
